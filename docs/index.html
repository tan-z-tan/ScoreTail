Why ScoreTail

Music was never meant to be a solo act.

Traditional notation software was built for isolated desktops and professional engravers. ScoreTail is built for musicians who want to create, share, and enjoy music together.

01

Just open a browser

No downloads. No installation wizards. Open a link and you're in the editor. Works on any device with a browser.

02

Compose with keyboard

Press A–G for pitch, 1–6 for duration, Space to play. Input flows as naturally as playing an instrument.

03

Share a link, not a file

Every score has a URL. Send it to a bandmate, a student, or post it on social media. MusicXML import/export when you need it.

Real-time Collaboration

Every edit, together.

Built on CRDT — the same technology behind collaborative text editors. No conflicts, no "your turn."

Simultaneous editing

Multiple people write notes on the same score at the same time. Every change appears instantly on everyone's screen.

Color-coded cursors

See exactly where each collaborator is working. No stepping on each other's toes.

One link to invite

No accounts, no invitations, no permissions setup. Share the URL and start composing together.

MusicXML & ABC notation

MusicXML ABC

Import scores from MuseScore, Sibelius, Dorico, or any editor that exports MusicXML. ABC notation is also supported for quick lead-sheet input. Export back anytime — your music is never locked in.
